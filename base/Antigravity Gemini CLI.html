¿Qué es Gemini CLI?

Una terminal conectada a Gemini. Nada más, nada menos.

01
definition 05 / 21
Definición

Un agente
en tu shell.

Gemini CLI es una herramienta open-source que trae el modelo Gemini directo a tu terminal — con acceso a tus archivos, a tu git y a tu entorno real.

Apache 2.0 Node 18+ Gemini 2.5 Pro
~/projects — gemini
$ gemini
Connected to gemini-2.5-pro · 1M context
› Refactoriza UserService.ts para usar Result<T>
› Leyendo 3 archivos…
› Propuesta generada (diff + tests)
$

Los que vas a usar.

prefixes 13 / 21

Tres prefijos. Tres mundos.

slash
/

Le hablas al CLI

Control interno: guardar chat, limpiar contexto, cambiar modelo, MCP tools.

at
@

Le hablas a tu repo

Referencia archivos o directorios. Gemini los lee antes de responder.

bang
!

Le hablas al shell

Ejecuta comandos del sistema sin perder el contexto de la sesión.

tips.md 19 / 21

Lo que
no sale en los docs.

tip · 01

GEMINI.md en el root.

Convenciones del repo, stack, coding style. Gemini lo lee automáticamente.

tip · 02

Checkpoints antes de editar.

/chat save pre-refactor para poder volver si algo explota.

tip · 03

Usá Flash para iterar.

Pro para tareas pesadas, Flash para Q&A. Se cambia con /model.

tip · 04

Compone con pipes.

git diff | gemini -p "resumí este PR" — es Unix, tratalo como tal.
